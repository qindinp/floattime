# FloatTime 更新日志

## v1.2.0 (2026-03-23)

### 新增功能
- 🎨 **日夜间模式切换**
  - 支持三种模式：跟随系统、日间、夜间
  - 自动模式：19:00-07:00 自动切换为夜间模式
  - 夜间模式使用深色背景，日间模式使用明亮背景
  
- ✨ **悬浮窗美化**
  - 圆角效果：18dp 圆角半径
  - 磨砂效果：95% 透明度背景
  - 阴影效果：8dp elevation
  - 根据主题和时间源动态调整颜色

- 📝 **运行日志**
  - 记录服务启动、停止、时间同步等操作
  - 日志文件保存在应用私有目录
  - 最多保留 100 行日志
  - 主界面显示最近 20 行日志

### 改进
- 优化悬浮球触摸响应
- 改进时间同步逻辑
- 增强错误处理和日志记录

### 技术细节
- 使用 GradientDrawable 实现圆角背景
- 使用 SharedPreferences 持久化设置
- 使用 android.util.Log 输出调试日志
